## Vue 过渡动画系统

Vue 提供了 `<Transition>` 和 `<TransitionGroup>` 两个内置组件来处理元素和组件的进入/离开动画。

### 基本用法

```vue
<template>
  <button @click="show = !show">切换</button>
  <Transition name="fade">
    <p v-if="show">Hello, Vue!</p>
  </Transition>
</template>

<style>
.fade-enter-active, .fade-leave-active {
  transition: opacity 0.3s ease;
}
.fade-enter-from, .fade-leave-to {
  opacity: 0;
}
</style>
```

### CSS 过渡类名

Vue 会在适当的时机为元素添加/移除以下 CSS 类名：

| 类名 | 时机 |
|------|------|
| `v-enter-from` | 进入动画开始前 |
| `v-enter-active` | 进入动画进行中 |
| `v-enter-to` | 进入动画结束 |
| `v-leave-from` | 离开动画开始前 |
| `v-leave-active` | 离开动画进行中 |
| `v-leave-to` | 离开动画结束 |

### JavaScript 钩子

除了 CSS 类名，还可以通过 JavaScript 钩子精确控制动画：

```vue
<Transition
  @before-enter="onBeforeEnter"
  @enter="onEnter"
  @after-enter="onAfterEnter"
  @leave="onLeave"
>
  <div v-if="show">内容</div>
</Transition>
```

> 💡 **Tip**: 使用 `mode="out-in"` 可以让旧元素先离开，新元素再进入，避免布局跳动。
